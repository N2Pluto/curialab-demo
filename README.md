## Getting Started

Installation

```bash
npm install
``` 

Environment, you can add Curia API:

```env
NEXT_PUBLIC_BASE_API=
```

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
